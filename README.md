# 易班一键打卡<br>
## 脚本由Python编写，适合有python基础的人阅读，bat版本适用于windows,Linux平台直接使用源码<br/>
* <strong>exe编译使用方法</strong>
> 1.注册ttshitu.com账号，替换YiBan.py中的获取验证码部分账号密码<br/>
> 2.安装pyinstaller后执行pyinstall --onfile main.py<br/>
> 3.在bat文件中添加一条记录，start main.exe <你的学号><br/>
> 4.运行bat文件
* <strong>源码直接使用方法</strong>
> 1.注册ttshitu.com账号，替换YiBan.py中的获取验证码部分账号密码<br/>
> 2.在bat文件中添加一条记录，start python main.py <你的学号><br/>
> 3.运行bat文件
